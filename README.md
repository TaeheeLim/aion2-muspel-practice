# 무스펠의 성배 — 구슬 색맞추기 연습기

아이온2 신규 성역 '무스펠의 성배'의 **구슬–테두리 색맞추기 패턴**을 연습하는 **단일 파일 웹 게임**입니다.

## 실행 방법

- **가장 간단**: `index.html` 더블클릭 → 브라우저에서 바로 실행 (설치·인터넷 불필요)
- 로컬 서버: 이 폴더에서 `python -m http.server 8731` 후 `http://localhost:8731/` 접속

## 기믹 규칙 (게임 설명 기반)

1. 보스 중심의 시계 위치(예: 4시)에서 **구슬 3개가 한 줄**로 천천히 보스로 들어옵니다. 게임 시작 시 **캐릭터는 선택한 시작 위치**에 배치됩니다.
2. 각 구슬 색은 무작위(**빨 / 초 / 파**).
3. 보스에는 **빨·초·파 원형 테두리가 3겹** 있습니다. 3색은 매 라운드 **독립 무작위(같은 색 중복 가능)**.
4. **맨 앞(보스에 가까운) 구슬 = 가장 바깥 테두리 색**, 2번째 = 중간, 3번째 = 안쪽 테두리 색과 같아야 정답.
5. **장판은 캐릭터를 따라다니며** 점점 진해지다가, **거의 다 차오르면(약 85%) 캐릭터에서 분리·고정**되고 2초 시점에 **펑** 터집니다(점선→실선으로 표시). 터지면 장판이 지나는 줄의 구슬 색이 **빨→초→파→빨→초…** 순서로 한 칸 바뀝니다.
   - 구슬 3개는 나타날 때 **모두 같은 색일 수 없습니다(최소 2색)**.
6. 장판은 **보스 → 캐릭터 → 바깥 끝**까지 이어지는 직선입니다. 길이는 보스와의 거리와 **무관**하게 항상 가장자리까지 닿고, 그 직선이 지나는 **한 줄 전체(최대 3개)**의 색을 바꿉니다. 플레이어는 **각도(어느 줄)만** 조절하며, 한 번에 **무조건 1줄만** 바뀝니다. (바꾸기 싫은 줄은 조준을 피함)
7. **진입 타이밍** — 장판이 **3번 터지면 1번째 구슬**, 이후 **2번 더(총 5번)에 2번째**, 다시 **2번 더(총 7번)에 3번째**가 진입합니다. 단, 임계 폭발 직후 바로 들어가는 게 아니라 색이 확정된 뒤 **약 1초 후**에 보스로 빨려 들어갑니다.
8. **1·4·7·10시**는 구슬이 **3개씩 두 줄(6구슬)**로 나옵니다. 두 줄은 각도가 살짝 달라 한 번의 장판은 한 줄만 덮으니, 좌우로 미세 조정해 번갈아 처리합니다. (두 줄은 같은 진입 스케줄을 공유)
9. **잘못된 색 구슬이 2개 들어가면 게임 실패**입니다.
10. 등장하는 색 조합은 **반드시 3·5·7회 폭발 안에 테두리 색으로 맞출 수 있는 것만** 나옵니다(불가능한 조합은 자동 재추첨). 두 줄은 한 번에 한 줄만 칠할 수 있어 더 어렵습니다.

## 조작 / 화면

- **이동만** 하면 됩니다: `W` `A` `S` `D` / 방향키, 또는 마우스·터치로 끌기
- 장판은 캐릭터 각도를 따라 **자동 생성**(2초 주기) — 어느 줄을 칠지(또는 피할지)만 위치로 결정
- 화면 상단에서 **시작 위치(시계)** 만 선택합니다. (1·4·7·10시는 자동으로 두 줄)
- 좌상단에 테두리 타깃(바깥→안쪽 = 1·2·3번 구슬 색), 보스 아래에 `폭발 N/7` 표시

## 참고

실제 인게임 좌표·속도·구슬 개수를 1:1로 측정한 것이 아니라, 제공된 기믹 설명을 토대로 구현한 연습기입니다.
세부 수치는 `index.html` 상단에서 쉽게 조정할 수 있습니다:
`COMMIT`(진입 스케줄 `[3,5,7]`), `ENTER_DELAY`(진입 지연 1초), `ARM_TIME`(장판/주기 = 게임 전체 속도, 현재 3초),
`LOCK_FRAC`(장판 분리 시점 0.85), `BEAM_WF`(장판 폭), `DOUBLE_CLOCKS`(두 줄 위치), `spawnLane()`/`laneSolvable()`.
